
# Developer Portfolio

A modern, responsive full-stack developer portfolio showcasing projects, skills, and contact information with a functional backend API.

##  Tech Stack

### Frontend
- **HTML5** - Semantic markup
- **CSS3** - Responsive design & animations
- **JavaScript (ES6+)** - Dynamic functionality

### Backend
- **Node.js** - Runtime environment
- **Express.js** - Web framework
- **MongoDB** - Database
